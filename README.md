# Trabajo Final de Técnicas para la Ciencia de Datos
**Proyecto:** 

Técnica de Clasificación No Supervisada de las Comunidades Autónomas de España según los Niveles de Riesgos de Contaminación Atmosférica 

**Autoras:** 

* Patricia Contreras 
* María Claudia Angulo


**Curso:** 

Técnicas para la Ciencia de Datos


**Máster:** 

Máster en Ciencia de Datos y Aprendizaje Automático – Universidad de La Rioja


**Descripción:**

Este proyecto aplica técnicas de análisis y clasificación para evaluar la calidad del aire en España, utilizando datos oficiales de la European Environment Agency (EEA). Se clasifican las Comunidades Autónomas en tres niveles de contaminación (regular, desfavorable y muy desfavorable), según los límites internacionales establecidos por la OMS en 2021.
