# Trabajo Final de Técnicas para la Ciencia de Datos
Proyecto: Clasificación de los niveles de riesgos de contaminación atmosférica en las Comunidades Autónomas de España según PM2.5, NO2 y O3 utilizando los límites OMS 2021 (2018–2022).


Autoras: Patricia Contrera - María Claudia Angulo


Curso: Técnicas para la Ciencia de Datos


Máster: Máster en Ciencia de Datos y Aprendizaje Automático – Universidad de La Rioja (UNIR)


Descripción:
Este proyecto aplica técnicas de análisis y clasificación para evaluar la calidad del aire en España, utilizando datos oficiales de la European Environment Agency (EEA). Se clasifican las Comunidades Autónomas en tres niveles de contaminación (bajo, medio y alto), según los límites internacionales establecidos por la OMS en 2021.
